# Todo List

> A simple web todo list with RWD and good looking.

## Get Started
- Step1: install npm packages 
  ```bash
     npm install
  ``` 

- Step2: run webpack development server
  ```bash
     npm run dev
  ``` 

- Step3: open browser on http://localhost:8080 and enjoy!
